# 第二阶段范围说明：小米智能穿戴设备扩展

## 1. Sprint

**AJ15XIAO Sprint 2：智能穿戴设备扩展**

## 2. 首个任务

**AJ15XIAOMI-18：冻结16款穿戴设备名单及官方来源**

## 3. 阶段目标

在现有小米耳机智能选购与客服助手基础上，新增小米手环和智能手表两个产品类别，为后续字段设计、数据收集、向量知识库接入、产品对比和场景推荐提供统一范围。

## 4. 本任务范围

- 智能手环：8款
- 智能手表：8款
- 总计：16款
- 数据来源：仅允许使用 Xiaomi / REDMI 官方产品页、官方规格页、官方说明书或官方支持页
- 地区规则：以 `source_region` 标明资料地区，后续不得混用不同地区版本参数

## 5. 冻结规则

1. 每款产品使用唯一 `product_id`。
2. 官方产品名称不得自行翻译或简写。
3. 颜色、表带、材质版本不作为独立产品。
4. NFC版与非NFC版原则上不重复计数；确需区分时应作为 `variant` 处理。
5. 每款产品至少保留一个可访问的小米官方来源。
6. 找不到的参数后续使用 `null`，不得依据第三方测评猜测。
7. 只有 `leader_approved=true` 的产品才能进入正式数据收集。
