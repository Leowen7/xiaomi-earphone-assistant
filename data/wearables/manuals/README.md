# P2-14 穿戴设备基础问答原始资料

## 任务说明

本目录用于保存智能手环和智能手表的基础操作问答原始资料，为P2-6文本切分和P2-7 FAISS向量库构建提供输入数据。

本任务不重新收集P2-5中的结构化产品参数，只补充连接、充电、系统操作、健康监测、通知、表盘和常见故障等基础问答内容。

## 数据范围

- 智能手环：B01—B08，共8款
- 智能手表：W01—W08，共8款
- 产品总数：16款
- 每款问答数量：6条
- 问答总数：96条

## 原始文件

`raw/P2-14_Wearable_FAQ_Compatible.docx`

文档中的每条问答包括：

- 产品ID
- 产品名称
- QA ID
- 问题主题
- 用户问题
- 中文整理答案
- 小米官方来源链接
- 数据状态

## 数据来源

仅使用以下官方来源：

- 小米官方产品页面
- 小米官方FAQ
- 小米官方支持页面
- 小米官方用户指南

不使用第三方论坛、电商评论、自媒体教程或无来源的AI生成内容。

## 后续处理要求

P2-6负责人需要：

1. 从Word文档中提取96条基础问答；
2. 每个问题生成一个独立文本块；
3. 标记 `chunk_type=manual_faq`；
4. 保留产品ID、QA ID、主题、问题、答案和来源链接；
5. 与原有144条参数文本块合并；
6. 校验产品ID、QA ID和chunk ID唯一性；
7. 抽查官方来源与产品型号是否对应；
8. 校验通过后将数据状态由 `draft` 修改为 `approved`。
